### 캐싱 설정, 테스트 코드
- myblog.WebMvcConfig: Spring Boot에서 캐싱, ETag 설정

- Helper가 사용된 곳은 src/main/resources/templates의 include/header.html에서 확인 가능합니다. 
- myblog.web.StaticResourcesTest: 테스트 코드를 활용해 ETag 학습할 수 있어요.
